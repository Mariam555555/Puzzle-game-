🌟 **Overview**
The AI-Powered 8-Puzzle Solver is an interactive desktop application built with Python. It allows users to play the classic sliding puzzle game across different difficulty levels, while also featuring an automated AI solver that calculates and animates the optimal path to the winning state.

✨**Key Features**
Modern User Interface: Designed a sleek, dark-themed GUI using CustomTkinter, featuring rounded buttons, responsive layout, and intuitive level selection.

AI Solver Integration: Implemented state-space search algorithms (such as A* Search with Heuristics) to autonomously find the shortest possible solution for complex board configurations.

Interactive Gameplay: Users can manually navigate the puzzle using keyboard arrow keys, making it a fully playable game.

Real-time Analytics: Includes a live move counter and a built-in timer to track the user's or AI's performance.

Dynamic Difficulty: Features multiple starting configurations (Easy, Medium, Hard) to test both the player and the algorithm.
